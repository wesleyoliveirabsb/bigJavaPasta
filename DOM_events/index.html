Os eventos permitem com que a partir de certas ações você ative algumas funções dentro do site, um dos eventos mais famosos é o do tipo click

para inserir ele você precisa digitar dentro de alguma coisa onclick e vincular nele uma função como na div acima

<div> id="principal" onclick="clicar()"> Essa é a minha div </div>

isso vai fazer com que um click execute a função clicar, mas o que faz a função clicar?

function clicar(){ principal.style.background = 'blue' principal.innerText = 'Agora é azul' }

isso vai fazer com que ao clicar nessa div mude a cor e o texto dela. também é possivel mudar as coisas ao entrar em cotato com o mouse

isso é chamado de onmouseenter

também é necessário adicionar o parâmetro a tag da seguinte forma

<div> id="principal" onclick="clicar()"> Essa é a minha div </div>

e igualmente ao onclick também temos que criar uma função que faz ela funcionar (c:)

function entrar(){ principal.style.background ='green' principal.innerText = 'agora é verde' }
