Esse site foi criado para explicar o que é a árvore dom

toda árvore dom começa pelo window, essa sendo a janela do site

dentro dessa window temos acesso a várias coisas, a mais importante dela no momento sendo o document, que dá acesso ao html

a árvore dom é basicamente uma árvore genealogiaca de todo o site

manipulando usando o DOM

para realisar manipulações pelo código é preciso ter uma tag de <script>

a frase acima não foi escrita usando html nem css, a frase foi escrita usando a manipução dom via javascript com o seguinte código

<script> window.document.write('minha primeira manipulação com DOM') </script>

como podemos ver, primeiro acessamos o primeiro nivel da árvore sendo ele o window, logo em seguida acessamos o segundo nivel da árvore, nesse caso, o document.

com acesso ao document já podemos mudar coisas dentro do site via javascript

alertas

você viu um alerta quando abriu essa página, esse alerta também foi feito usando manipulação DOM, com o código

<script> window.alert('Oi!') >/script>

o alerta foi feito usando o código acima, acessando a window e já diretamente podemos modificar coisas dentro

Seletores

alem de acrescentar elementos novos podemos editar os elementos já existentes

usando a função getElementById do javascript podemos selecionar qualquer elemento selecionando ele pela id

nesse caso vamos criar uma variável com o código "var x = window.document.getElementById('principal')".

dessa forma sempre que utilizarmos a variável "x" estaremos nos referindo ao elemento com a id principal, nesse caso o elemento com a id principal é o body

a cor inicial dessa página era cinza, mas ela foi modificada usando o javascript para a aparencia de agora com o seguinte código

<script> var x = window.document.getElementById('principal') x.style.backgroundColor='black' >/script>

também podemos selecionar elementos pelo name, e até mesmo pela tagname.
o que é uma tagname? é o nome das tags usadas no html como h1, p , ou button.

mas como podemos ter várias tags iguais, para selecionar especificamente devemos indicar qual o numero da tag usando o [] e começando pelo zero

var y= window.document.getElementsByTagName('button')[0]
